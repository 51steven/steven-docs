# 一、常用的性能评价/测试指标 

## 1.1 响应时间

提交请求和返回该请求的响应之间使用的时间，一般比较关注平均响应时间。如：数据库查询花费的时间，将字符回显到终端上花费的时间，访问 Web 页面花费的时间。
